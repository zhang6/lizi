# 3D 交互体验集合

一个使用 Three.js 和 MediaPipe 创建的交互式 3D 体验集合，包含多个精彩的 WebGL 作品。

## 在线体验

点击这里立即体验：[https://airfx.yesxno.com](https://airfx.yesxno.com)

## 项目列表

### 🎄 Grand Luxury Tree
精美的 3D 圣诞树场景，采用豪华设计风格，展现华丽的视觉效果和流畅的交互体验。

**功能特性：**
- 🎄 3D 圣诞树场景，金色装饰和灯光效果
- 🖼️ 上传自己的照片展示在相框里
- 👋 手势识别控制（摄像头）
- 🎨 多种交互模式（TREE、SCATTER、FOCUS）
- 🎯 可旋转查看场景

**使用说明：**
1. **允许摄像头权限**：首次使用需要允许浏览器访问摄像头
2. **手势控制**：
   - 🤏 **捏合手势**：拇指和食指捏合
     - 选择距离手最近的照片并放大聚焦显示，其他粒子散开
   - ✊ **收拢手势**：手指收拢（握拳或半握）
     - 粒子聚集成圣诞树形状，树会缓慢旋转
   - ✋ **张开手势**：手指张开
     - 粒子散落成3D球体分布，粒子会旋转
   - 👈👉 **左右滑动**：在张开手势模式下，手左右移动
     - 控制3D场景的水平旋转（Y轴旋转），轻轻滑动即可产生大幅旋转
   - 👆👇 **上下滑动**：在张开手势模式下，手上下移动
     - 控制3D场景的垂直旋转（X轴旋转）
3. **上传照片**：点击 "Add Memories" 按钮上传图片
4. **隐藏控制**：按 `H` 键隐藏/显示控制按钮
5. **查看手势说明**：点击 "📖 Guide" 按钮查看详细的手势说明面板
6. **摄像头窗口**：点击 "📷 Camera" 按钮或按 `C` 键显示/隐藏摄像头窗口，实时查看手势识别效果

### ✨ 3D 粒子手势交互系统
基于 MediaPipe 的手势识别系统，通过手势操控 3D 粒子效果，体验前沿的人机交互技术。

**功能特性：**
- ✨ 45000+ 粒子的 3D 效果
- 👋 手势控制：五指张合可以放大缩小，手掌离近离远也能调节效果
- 🔄 双指旋转：平面翻转（旋钮操控）
- 👆 食指滑动：视角旋转（锁定缩放）
- 🎨 可切换多种粒子形状（球体、星系、土星等）
- 🌈 自定义粒子颜色

**使用说明：**
1. **允许摄像头权限**：首次使用需要允许浏览器访问摄像头
2. **手势控制**：
   - 👋 **五指张合**：控制粒子缩放（张开放大，合拢缩小）
   - 👆 **食指滑动**：旋转视角
   - ✌️ **双指旋转**：平面翻转
3. **切换形状**：点击右侧按钮切换粒子形状
4. **调整颜色**：点击颜色选择器调整粒子颜色

## 技术栈

- **Three.js** - 3D 图形渲染（WebGL）
- **MediaPipe** - 手势识别和手部追踪
- **WebGL** - 底层图形渲染 API

## 系统要求

### 浏览器兼容性
- ✅ Chrome/Edge（推荐）
- ✅ Firefox
- ✅ Safari（Mac）
- ⚠️ 需要支持 WebGL 和摄像头访问

### 硬件要求
- **显卡**：支持 WebGL 的现代显卡（不限于 NVIDIA）
  - ✅ NVIDIA 独立显卡
  - ✅ AMD 显卡
  - ✅ Intel 集成显卡
  - ✅ Apple Silicon（M1/M2/M3 等）
- **摄像头**：用于手势识别功能
- **性能**：建议使用现代设备以获得最佳体验

### Mac 兼容性
- ✅ **Apple Silicon（M1/M2/M3）**：完全支持，性能优秀
- ✅ **Intel Mac**：支持，可能使用 CPU 模式运行 MediaPipe，性能稍慢但仍可正常使用

## 本地运行

1. 克隆仓库：
```bash
git clone <repository-url>
cd christmas-tree
```

2. 使用本地服务器运行（推荐）：
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```

3. 在浏览器中访问 `http://localhost:8000`

> ⚠️ 注意：由于使用了 ES 模块和摄像头 API，需要通过 HTTP 服务器运行，不能直接打开 HTML 文件。

## 部署说明

本项目使用 GitHub Actions 自动部署到 GitHub Pages。

### 首次部署步骤：

1. 在 GitHub 仓库 Settings -> Pages 中：
   - Source 选择 **"GitHub Actions"**
   - 保存

2. 推送代码后，GitHub Actions 会自动部署

3. 访问 `https://github.com/jingkeling/airfx`
